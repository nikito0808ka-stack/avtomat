## **Тип автомата**: , вероятностный автомат.
  ### Вероятностный автомат — это математическая модель вычислительного устройства или системы с конечным числом состояний, в которой переходы между состояниями происходят не детерминированно, а с определённой вероятностью.
  ## **Ключевые особенности**
### - Вероятностные переходы. Для каждой пары (текущее состояние, входной символ) задаётся не одно следующее состояние, а распределение вероятностей по возможным следующим состояниям.
### - Сумма вероятностей. Для любого состояния и входного символа сумма вероятностей всех возможных переходов из этой пары равна 1:
  ### Как работает ДКА
   ### 1) Автомат начинает работу в начальном состоянии q0
   ### 2) Последовательно считывает символы входной строки.
   ### 3) Для каждого символа, используя функцию переходов , переходит в следующее состояние.
   ### 4) После обработки всей строки:
 ### - Eсли автомат оказался в одном из состояний множества F, то строка принимается;
 ### - В противном случае строка отклоняется.
