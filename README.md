## **Тип автомата**: детерминированный конечный автомат (ДКА),
  ### Это математическая модель, которая принимает или отклоняет заданную строку символов путём прохождения через последовательность состояний, строго определённую входной строкой
  ### Как работает ДКА
   ### 1) Автомат начинает работу в начальном состоянии q0
   ### 2) Последовательно считывает символы входной строки.
   ### 3) Для каждого символа, используя функцию переходов , переходит в следующее состояние.
   ### 4) После обработки всей строки:
 ### - Eсли автомат оказался в одном из состояний множества F, то строка принимается;
 ### - В противном случае строка отклоняется.

## **Множество состояний**: Q = {q0, q1, q2 , q3},

## **Входной алфавит**: Σ = {a, b},

## **Начальное состояние**: q0,

## **Конечные состояния**: F = {q1​}.
